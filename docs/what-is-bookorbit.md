# What is BookOrbit?

BookOrbit is a self-hosted library and reading platform. You run it on your own server to organize books, read across formats, sync progress, and manage users without relying on a third-party cloud account.

No subscription. No vendor lock-in. Your library stays on your infrastructure.

## What you get

- **Built-in readers** for ebooks, PDFs, comics, and audiobooks
- **Library management** with multiple libraries, collections, smart scopes, and series support
- **Metadata automation** with provider-based enrichment and field-level control
- **Reading analytics** for sessions, progress, pace, and library trends
- **Multi-user access** with permissions, per-library access control, and isolated progress data

## Format support

| Reader | Formats |
|--------|---------|
| **eBook** | EPUB, KEPUB, MOBI, AZW3, AZW, FB2 |
| **PDF** | PDF |
| **Comics** | CBZ, CBR, CB7 |
| **Audiobook** | M4B, MP3, M4A, OPUS, OGG, FLAC |

## Works with your devices

| Integration | Use case |
|-------------|----------|
| **Kobo sync** | Sync supported books and reading position with Kobo devices |
| **OPDS** | Connect reader apps over a private OPDS catalog |
| **Email delivery** | Send files directly to Kindle or other email-based workflows |
| **Book Dock** | Stage uploads, validate metadata, then finalize into libraries |

## Built for self-hosting

BookOrbit runs as a Docker deployment with a standard database backend. It keeps your existing folder structure, processes metadata and search locally, and avoids external service dependencies unless you explicitly configure them.
